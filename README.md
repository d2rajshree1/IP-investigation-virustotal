# IP Investigation using VirusTotal

## 📌 Objective
To demonstrate the investigation of a public IP address using VirusTotal.

## ⚠️ Note
Initial analysis using netstat did not reveal any malicious IPs. Therefore, a known public IP was selected to simulate the investigation process.

## 🛠 Tools Used
- netstat
- VirusTotal

## 🔍 Analysis Includes
- Detection Overview
- ASN, ISP, Location
- Reputation
- Relations
- Timeline

## 📂 Project Structure
- report.md → Detailed findings
- screenshots/ → Evidence images
